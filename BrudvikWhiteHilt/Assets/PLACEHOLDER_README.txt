PLACEHOLDER ICONS
=================

This folder should contain PNG icons for all potions.
The icons are embedded resources and must be added to the .csproj file.

Required icons (32x32 or 64x64 PNG recommended):
- GiftOfSleipnir.png
- GiftOfRatatoskr.png
- GiftOfNjord.png
- GiftOfSurt.png
- GiftOfSkadi.png
- GiftOfBaldur.png
- GiftOfThor.png
- GiftOfIdunn.png
- GiftOfBrokkr.png
- GiftOfTyr.png
- GiftOfFenrir.png
- GiftOfFreyr.png
- GiftOfHel.png
- GiftOfMimir.png

Note: The original icons were purchased from Graphicriver.net and cannot be redistributed.
You need to either:
1. Purchase your own license from https://graphicriver.net/item/rpg-potion-icons/24972053
2. Create your own icons
3. Use free/open-source icons

Each icon should be added to the .csproj as an EmbeddedResource:
<EmbeddedResource Include="Assets\GiftOfSleipnir.png" />
